1 The Agentâ€“Environment Interface

  • \(t=0,1,2,3,\cdots\), at each time step \(t\), the agent receives some representation of the environment's state \(S_t\in \mathcal{S}\), where \(\mathcal{S}\) is the set of possible states, and on that basis selects an action \(A_t\in \mathcal{A}(S_t)\).. where \(\mathcal{A}(S_t)\) is the set of actions available in state \(S_t\). one time step later, in part as a consequence of its action, the agent receives a numberical reward, \(R_{t+1}\in \mathcal{R}\subset \mathbb{R}\), and finds itself in a new state, \(S_{t+1}\)
  • policy : \(\pi_t(a|s)\) is the probability that \(A_t=a\) if \(S_{t}=s\)

2 Returns

  • \(G_t\doteq R_{t+1}+\gamma R_{t+2}+\gamma^2R_{t+3}+\cdots=\sum_{k=0}^{\infty}\gamma^k R_{t+k+1}\)
  • \(G_t\doteq \sum_{k=0}^{T-t-1}\gamma^kR_{t+k+1}\)

3 Markov Decision Processes

A particular finite MDP is defined by its state and action sets and by the one-step dynamics of the environment. Given any state and action \(s\) and \(a\), the probability of each possible pair of next state and reward, \(s'\),\(r'\), is denoted

\begin{equation} \label{eq:1} p(s',r|s,a)\doteq Pr\{S_{t+1}=s',R_{t+1}=r|S_t=s,A_{t}=a\} \end{equation}

expected rewards for stateâ€“action pairs

\begin{equation} \label{eq:2} r(s,a)\doteq \mathbb{E}[R_{t+1}|S_t=s,A_t=a]=\sum_{r\in \mathcal{R}}\sum_{s'\in \mathcal{S}}p(s',r|s,a) \end{equation}

the state-transition probabilities

\begin{equation} \label{eq:3} p(s'|s,a)\doteq Pr\{S_{t+1}=s'|S_{t}=s,A_{t}=a\}=\sum_{r\in \mathcal{R}}p(s',r|s,a) \end{equation}

expected rewards for stateâ€“actionâ€“next-state triples

\begin{equation} \label{eq:4} r(s,a,s')\doteq \mathbb{E}[R_{t+1}|S_t=s,A_t=a,S_{t+1}=s']=\frac{\sum_{r\in \mathcal{R}}rp(s',r|s,a)}{p(s'|s,a)} \end{equation}

4 Value Functions

Recall that a policy, \(\pi\), is a mapping from each state, \(s\in \mathcal{S}\), and action, \(a\in \mathcal{A}(S)\), to the probability \(\pi(a|s)\) of taking action \(a\) when in state \(s\). Informally, the value of \(a\) state \(s\) under a policy \(\pi\), denoted \(\nu_{\pi}(s)\), is the expected return when starting in \(s\) and following \(\pi\) thereafter. For MDPs, we can define \(\nu_{\pi}(s)\) formally as

\begin{equation} \label{eq:5} \nu_{\pi}(s)\doteq \mathbb{E}_{\pi}[\sum_{k=0}^{\infty}\gamma^kR_{t+k+1}|S_t=s] \end{equation}
  • \(\mathbb{E}_{\pi}\) : the expected value of a random variable given that the agent follows policy \(\pi\) and \(t\) is any time step.

We call the function \(\nu_{\pi}\) the state-value function for policy \(\pi\).

Similarly, we define the value of taking action \(a\) in state \(s\) under a policy \(\pi\), denoted \(q_{\pi}(s,a)\), as the expected return starting from \(s\), taking the action \(a\), and thereafter following policy Ï€:

\begin{equation} \label{eq:6} q_{\pi}(s,a)\doteq \mathbb{E}_{\pi}[G_t|S_t=s,A_t=a]=\mathbb{E}_{\pi}[\sum_{k=0}^{\infty}\gamma^kR_{t+k+1}|S_t=s,A_t=a] \end{equation}

We call \(q_{\pi}\) the action-value function for policy \(\pi\).

\begin{eqnarray} \label{eq:8} \nu_{\pi}(s) & \doteq & \mathbb{E}_{\pi}[G_t|S_t=s]\\ &=&\mathbb{E}_{\pi}[R_{t+1}+\gamma G_{t+1}|S_t=s]\\ &=&\sum_a\pi(a|s)\sum_{s'}\sum_r p(s',r|s,a)[r+\gamma \mathbb{E}_{\pi}[G_{t+1}|S_{t+1}=s']]\\ &=&\sum_a\pi(a|s)\sum_{s',r}p(s',r|s,a)[r+\gamma \nu_{\pi}(s')] \end{eqnarray} \begin{equation} \label{eq:12} q_{\pi}(s,a)=\mathbb{E}_{\pi}[R_{t+1}+\gamma q(S_{t+1},A_{t+1})|S_t=s,A_t=a] \end{equation}

\begin{equation} \label{eq:14} \nu_{\pi}(s)=\sum_{a\in \mathcal{A}}\pi(a|s)q_{\pi}(s,a) \end{equation} \begin{equation} \label{eq:15} q_{\pi}(s,a)=R_s^a+\gamma\sum_{s'}P_{ss'}^a\nu_{\pi}(s') \end{equation}

5 Optimal Value Functions

optimal state-value function

\begin{equation} \label{eq:9} \nu_{*}(s)\doteq \max_{\pi}\nu_p(s) \end{equation}

optimal action-value function

\begin{equation} \label{eq:10} q_{*}(s,a)\doteq \max_{\pi}q_{\pi}(s,a) \end{equation} \begin{equation} \label{eq:11} \nu_{*}(s)=\max_a R_s^a+\gamma \sum_{s'\in \mathcal{S}}P_{ss'}^{a}\nu_{*}(s') \end{equation} \begin{equation} \label{eq:16} q_{*}(s,a)=R_s^a+\gamma\sum_{s\in \mathcal{S}}P_{ss'}^a\max_{a'}q_{*}(s',a') \end{equation}
